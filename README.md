# Grade Calculator

A simple grade calculator app created with Javascript, HTML/CSS.
Users can calculated their mean grade or weighted grade, and add as many entries as needed.




## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku CLI](https://cli.heroku.com/) installed.

```sh
$ git clone https://github.com/gracelu0/calculator.git # or clone your own fork
$ cd calculator
$ npm install
$ npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
